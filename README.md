#  Iniciar Página

Para levantar el servidor del proyecto es necesario tener instalado **npm**.  
Sigue los pasos a continuación:

## 1 Entrar a la carpeta del frontend
```bash
cd frontend-gimnasio
```
## 2 Instalar dependencias
```bash
npm install
```
## 3 Iniciar el servidor
```bash
npm run dev
```
Al ejecutar este comando, se desplegará una dirección IP en la terminal.
Para abrir la aplicación en tu navegador, presiona Ctrl + click sobre la dirección IP mostrada.
